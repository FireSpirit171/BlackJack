## Внимание!

В папке `project` лежит файл `BlackJack.py` - исходный код проекта. Для запуска игры необходимо иметь версию Python 3.11 и выше. 
Для компиляции файла `.py` в `.exe` можно воспользоваться утилитой `auto-py-to-exe`
